Student Analyzer
Project Description

This project is a simple Student Analyzer written in Python.
It reads student data from a CSV file, calculates statistics, assigns grades, and displays results using both a command-line interface and a graphical user interface (GUI).

Features
Load student data from students.csv
Calculate average, highest, and lowest scores
Assign letter grades (A, B, C, D)
Display data in a table
Show charts using matplotlib
Simple GUI built with Tkinter
Project Files
gui_app.py        # GUI program
main.py           # command-line version
analyzer.py       # calculations
students.csv      # data file
test_analyzer.py  # unit tests
How to Run
Run GUI version
python gui_app.py
Run command-line version
python main.py
Run tests
python test_analyzer.py
